# Indian Unbanked District Classifier

## Overview

An AI/ML-powered web application that identifies underserved and unbanked districts across India based on population, area, and banking infrastructure.

The project combines district-level demographic and banking data, applies data preprocessing and feature engineering, and classifies districts according to banking accessibility levels.

## My Contribution

- Cleaned and standardized the raw data.
- Merged multiple datasets into a unified analytical dataset.
- Performed feature engineering and accessibility metric calculations.
- Developed the district classification framework.
- Built an interactive Streamlit dashboard.

## Dataset Sources

Data was compiled from publicly available government and banking-related sources, including:

- Census of India - https://censusindia.gov.in/census.website/data/census-tables
- Reserve Bank of India (RBI)- data.gov.in / RBI banking infrastructure datasets
